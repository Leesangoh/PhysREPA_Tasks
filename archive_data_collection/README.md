# Archived data collection and simulation code

This directory stores historical code that is not part of the active PEZ-to-PhysProbe probing workflow.

It includes:

- Isaac Lab environment configs
- RL environment wrappers
- scripted policies
- rollout / evaluation / recording utilities
- dataset collection scripts
- verification scripts
- earlier probing and shuffle-analysis results
- the previous `archive_v1/` snapshot

The archive was created to keep the repository root focused on probing work only. Nothing here is deleted; it is preserved for reference and recovery if a future experiment needs the original collection or simulation pipeline again.
